# Repair Center POS System

A full-stack point-of-sale system built for phone and computer repair shops. This project is designed to support real-world repair workflows by combining ticket management, inventory tracking, customer records, technician metrics, authentication, and payment integration into one scalable platform.

## Overview

The Repair Center POS System is a production-style application focused on improving efficiency for repair businesses. It is being built to help technicians, managers, and owners manage daily operations through a clean, responsive interface that works across desktop and tablet devices.

This project is especially focused on real business needs, including secure role-based access, workflow visibility, and scalable system design.

## Features

- Repair ticket creation and tracking
- Customer record management
- Inventory and parts tracking
- Technician performance metrics
- Role-based authentication and authorization
- Payment integration with Square
- Responsive UI for desktop and iPad/tablet use
- Scalable relational database design
- REST API architecture for backend services
